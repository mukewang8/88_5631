<h3>课程介绍：</h3>
<p>程围绕电商项目大觅网的业务场景，基于微服务原则设计电商项目，使用多种技术如Eureka、Feign、Hystrix、Ribbon、Zuul、Config等，另外基于虚拟化技术Docker+Jenkins实现程序自动发布、基于Mycat实现第三方支付接入、整个项目的高并发测试等。</p>
<p>课程以实战为基础，让同学们在实战过程中，独立完成网站的架构搭建和项目开发，掌握其中的实现方式与思路。</p>
